# Java Spring Boot 個人開発テンプレート

クリーンアーキテクチャに基づいた Spring Boot + PostgreSQL の個人開発用テンプレートリポジトリです。

このリポジトリは「Use this template」ボタンから新しいプロジェクトの出発点として利用できます。

## 構成

```
.
└── template/    # Spring Boot プロジェクト本体
```

## 始め方

プロジェクト本体は `template/` 配下にあります。詳細なセットアップ手順、技術スタック、アーキテクチャの説明は [`template/README.md`](./template/README.md) を参照してください。

設計判断の記録は [`template/docs/ARCHITECTURE.md`](./template/docs/ARCHITECTURE.md) にまとめています。
